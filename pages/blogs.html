What is the difference between Arrow Function and Regular Function

While both regular JavaScript functions and arrow functions work in a similar manner, there are certain differences between them.such as: 1. Syntax 2. Arguments binding 3. this 4. new keyword 5. No duplicate named parameters 6. Function Hoisting 7. Methods
